 📱 FlashChat iOS

Production-style iOS chat application built with **Swift**, **UIKit**, and **Firebase**.  
Implements real-time messaging with a clean UI inspired by modern messengers.

---

## 🚀 Features

- 🔐 User authentication (Login / Register)
- 💬 Real-time messaging (Firebase Firestore)
- 🎨 Custom chat UI (iMessage-style bubbles)
- ⚡ Smooth message animations
- 📲 Auto-scrolling chat
- ⌨️ Keyboard handling
- 🧠 MVVM architecture

---

## 🛠 Tech Stack

- Swift
- UIKit
- Firebase Auth
- Firebase Firestore
- Auto Layout
- MVVM

## ⚠️ Setup

This project uses Firebase.

**Important:**  
Firebase configuration file is not included.

To run the app:
1. Create your Firebase project
2. Download `GoogleService-Info.plist`
3. Add it to the project

🧠 Architecture

The app follows MVVM pattern:
	•	ViewControllers → UI logic
	•	ViewModels → business logic
	•	Services → Firebase interaction
